Introduction au Système

Le système de pilotage d'électrification de masse est une solution complète pour gérer, suivre et optimiser les projets d'installation électrique à grande échelle. Il permet de coordonner les différents corps de métier, détecter les problèmes en temps réel et optimiser les ressources.

Fonctionnalités principales

  • • Planification automatisée des ressources
  • • Suivi en temps réel de l'avancement
  • • Détection automatique des goulots d'étranglement
  • • Simulation de scénarios
  • • Génération automatique de rapports
  • • Saisie terrain avec mode hors ligne

Véhicules et mode d'acquisition

La page Paramètres permet de préciser le nombre de véhicules par acteur et le mode d'acquisition (Acheter ou Louer).

  • Acheter : le coût d'achat par unité est pris en compte une seule fois dans les charges (coût d'investissement).
  • Louer : le coût de location journalier est multiplié par la durée estimée du projet et ajouté aux coûts opérationnels.
  • Les acteurs concernés : Chef de projet, Contrôleur, Installateur réseau et Camions de livraison.

Si vous ouvrez un ancien projet sans ces champs, des valeurs par défaut seront appliquées automatiquement pour maintenir la compatibilité.

Dashboard Principal

Le dashboard est votre vue d'ensemble du projet. Il affiche en temps réel tous les indicateurs clés nécessaires au pilotage.

Indicateurs clés

  • Avancement Global : Pourcentage total de progression
  • Ménages Terminés : Nombre de ménages traités
  • Équipes Actives : Nombre d'équipes en cours de travail
  • Jours Restants : Durée estimée jusqu'à la fin du projet

Graphiques

  • Avancement par métier : Visualisation de la progression par corps de métier
  • Productivité quotidienne : Évolution de la production journalière
  • Timeline du projet : Planning et échéancier détaillé

Astuce : Cliquez sur les éléments du tableau de bord pour obtenir des détails supplémentaires.

Configuration des Paramètres

La section Paramètres permet de configurer tous les aspects du projet. Les modifications sont appliquées en temps réel et sauvegardées automatiquement.

Paramètres du projet

  • • Nombre total de ménages (calculé automatiquement depuis les zones)
  • • Date de début du projet
  • • Distance moyenne entre ménages
  • • Taux d'imprévus

Découpage géographique

  • • Création de zones avec nombre de ménages
  • • Mode de répartition (Auto/Manuel)
  • • Allocation automatique des équipes par zone

✨ Configuration Optimisée (Nouveau !)

Le bouton "Configuration Optimisée" calcule et remplit automatiquement tous les champs de la page Paramètres en fonction de vos zones.

Comment ça marche ?
  1. Créez au moins une zone avec un nombre de ménages
  2. Le bouton "Configuration Optimisée" s'active automatiquement
  3. Cliquez sur le bouton
  4. Tous les champs sont remplis instantanément !

Logique de calcul automatique

Le système calcule le nombre optimal d'équipes par métier en fonction de :

  • • Productivité journalière : Ex. 3 murs/jour pour les maçons
  • • Nombre de ménages : Total de toutes vos zones
  • • Durée cible : 180 jours par défaut
  • • Formule : Équipes = ceil(ménages / (productivité × durée))

Exemple : Pour 1000 ménages, le système calcule automatiquement 2 équipes de maçons, 2 équipes réseau, 1 équipe intérieur Type 1, etc.

Guide : Obtenir un résultat (Pas à pas)

Ce guide rapide vous montre comment paramétrer, simuler et exporter un résultat exploitable en quelques étapes.

Étapes rapides

  1. Choisir un projet : Ouvrez `Paramètres` et vérifiez les données de base (nombre de ménages, date de début).
  2. Configurer les équipes : Définissez le nombre d'équipes et les taux de production par métier.
  3. Définir les scénarios : Dans `Simulation`, ajustez le taux d'imprévus, la stratégie de déploiement (rapide, économique, sécurisé).
  4. Lancer la simulation : Cliquez sur Lancer la simulation et attendez la fin du calcul.
  5. Analyser les indicateurs : Consultez `Dashboard` et `Indicateurs` pour voir l'avancement, les risques et les goulots d'étranglement.
  6. Exporter le rapport : Dans `Rapports`, choisissez le format (PDF/Excel) et générez le document.
  7. Appliquer / Synchroniser : Si le scénario est satisfaisant, appliquez la configuration et synchronisez la saisie terrain.

Exemple concret

Paramètres choisis : 1 000 ménages, 10 équipes, production 8 ménages/jour, taux d'imprévus 5%.

Résultat attendu après simulation : durée estimée ~13 semaines, coût estimé X, et goulots détectés dans la phase de livraison.

Checklist avant export

  • • Vérifier la période choisie pour le rapport
  • • Confirmer que les équipes et coûts sont à jour
  • • S'assurer que la simulation a convergé sans erreurs
  • • Valider les filtres de données (zones, métiers)

Dépannage rapide

  • • Simulation trop lente : réduisez la complexité du scénario ou activez le mode 'Déploiement Rapide'.
  • • Données manquantes : allez dans `Paramètres` et complétez les champs obligatoires.
  • • Erreur d'export : réessayez en changeant le format (ex. PDF → HTML) ou vérifiez la console du navigateur pour messages d'erreur.
  • • Mode hors-ligne : vérifiez la synchronisation une fois la connexion rétablie.

Questions Fréquentes

Comment détecter automatiquement les goulots d'étranglement ?

Le système analyse en permanence les vitesses d'avancement de chaque corps de métier. Un goulot d'étranglement est détecté quand un métier avance significativement plus vite ou plus lentement que les autres, créant un déséquilibre dans la chaîne de production.

Puis-je utiliser le système sans connexion internet ?

Oui, le système fonctionne en mode hors ligne. Les données sont sauvegardées localement et synchronisées automatiquement dès qu'une connexion est disponible. La saisie terrain peut se faire sans connexion.

Comment personnaliser les rapports ?

Vous pouvez choisir la période du rapport (mensuelle, trimestrielle, annuelle), le format d'export (PDF, HTML, Word, Excel) et le type de contenu (complet, exécutif, performance, risques).

Le système peut-il suggérer des réaffectations d'équipes ?

Oui, l'algorithme d'optimisation analyse les déséquilibres entre métiers et suggère des réaffectations d'équipes pour optimiser le flux global de production et réduire les temps d'attente.

Comment fonctionne la simulation de scénarios ?

Le simulateur permet de tester différentes configurations (nombre d'équipes, taux d'imprévus, etc.) et calcule automatiquement l'impact sur la durée du projet, les coûts, la productivité et les risques.
