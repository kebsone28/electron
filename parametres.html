Import / Export

Formats supportés : .xlsx, .xls, .csv. Les données seront automatiquement converties si possible.

Conseil : Les modifications sont sauvegardées automatiquement. Utilisez le bouton "Configuration Optimisée" pour des paramètres pré-calculés.

Importer Données Terrain & Cartographie Ménages

Importez un fichier CSV/JSON contenant les relevés terrain (maisons traitées, murs construits, etc.) et la cartographie des ménages (GeoJSON ou CSV avec lat/lon). Après import, la simulation peut se lancer automatiquement.

Format recommandé: GeoJSON features avec propriétés (household_id, other props) ou CSV avec colonnes lat,lon,id.

Colonnes attendues (CSV): date,team,housesCount,hoursWorked,status,remarkDescription. Les champs seront automatiquement convertis si possible.
